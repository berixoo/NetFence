视觉风格方案

基于侧栏导航架构，选择 NetFence 的整体视觉风格

A

Fluent Dark — 深色专业风

Windows 11 原生风格，深色背景+亚克力材质，蓝色强调色。与系统设置、终端等工具视觉统一，长时间使用不刺眼。

B

Fluent Light — 浅色清爽风

Windows 11 浅色主题，白色背景+轻阴影，信息密度高时阅读清晰。适合白天办公环境。

C

跟随系统（推荐）

自动检测 Windows 系统主题设置，深色/浅色自动切换。用户无需手动设置，体验最自然。
