NetFence 导航架构方案

当前为单窗口+标签页，未来需容纳：程序扫描、联网监控、服务/计划任务、规则档案、联网模式、日志设置。哪种导航模式最合适？

A

侧栏导航（Sidebar）

左侧固定图标+文字导航栏，右侧内容区切换。类似 VS Code / Windows 设置。模块再多也不拥挤，扩展性最好。

B

顶部标签页（Tabs）

保持当前标签页模式，顶部横向排列。简单直观，但 6+ 标签页会拥挤，需要折叠或滚动。

C

Ribbon 工具栏

类似 Office 的功能区模式，顶部大图标分组。适合操作密集型，但占用垂直空间大。
